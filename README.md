# LLM demo

1. install ollama
2. go into whatever model folder you want to use
3. run command `ollama create <name of the model> -f <MODELFILE_NAME> `
4. run the model:
    - install ollama app [linux](https://github.com/JHubi1/ollama-app/releases/download/1.2.0/ollama-linux-x64-v1.2.0.tar.gz) / [Windows](https://github.com/JHubi1/ollama-app/releases/download/1.2.0/ollama-windows-x64-v1.2.0.exe)
    - start the app (for linux go inside the downloaded and extracted folder and run ./ollama)
    - check the host on the app settings
